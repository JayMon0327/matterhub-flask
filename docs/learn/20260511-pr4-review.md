## 채택된 패턴 (스킬에서 추출)

- **의도 매칭 트랙과 LLM 트랙 완전 분리**: 명령 발화는 LLM 호출 자체를 우회 → LLM 토큰 누수가 디바이스 제어 정확도를 망치지 못함. 시연 안정성 핵심 설계 원칙
- **단일 HTTP 라인으로만 묶인 도메인 다리**: `matter.py` 한 파일이 효돌 ↔ matterhub-api 사이 유일한 결합점. timeout=3s + try/except로 감싸 한쪽 장애가 전파 안 됨
- **SSH heredoc 패치 금지**: 한국어 주석/escape 이중 풀림으로 SyntaxError 빈발 → 로컬 `.py` 작성 후 `scp` 전송 + 디바이스에서 실행 패턴 박제
