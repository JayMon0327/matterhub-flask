# MQTT 코나이 패치 스크립트 작성

## 이슈 요약

코나이측이 SSH로 장비에 접속하여 한 번 실행하면 MQTT 연결 안정성 패치가 적용되는 셀프서비스 스크립트 및 전달 문서 작성.

## 수정 내용

### 1. 원나잇 패치 스크립트 (`patch_mqtt_stability.sh`)
- 4개 파일(mqtt.py, runtime.py, publisher.py, state.py) 자동 패치
- Python 스크립트 기반 정밀 문자열 치환 (sed 대신 안전한 방식)
- 자동 백업 + 롤백 기능 (`--rollback`)
- dry-run 모드 (`--dry-run`)
- __pycache__ 정리 후 서비스 재시작
- 실행 결과 자동 검증 (서비스 상태 + 로그 tail)

### 2. 문제 원인 분석 문서
- 4가지 근본 원인 코드 레벨 상세 설명
- 증상-원인 매핑 테이블
- 해결 방안 요약

### 3. 패치 적용 가이드
- 한 줄 실행 방법
- 패치 전/후 예상 동작 비교 (실제 로그 형식)
- 검증 방법 (journalctl 명령어)
- 롤백 방법

## 배포 여부

- 미배포 (스크립트 + 문서 작성 완료, 코나이측 전달 대기)

- 패치 대상 코드 변경 검증 (Phase 1~4 테스트 전체 통과) ✅
