## 발견된 이슈

### 채굴 악성코드 (15027, 15065)
- `/tmp`에 `nvidia:nvidia` 소유 숨김 디렉토리 ~23,000개 (4KB씩, 총 ~199GB)
- `/var/tmp`에도 동일 패턴 (~197-200GB)
- 프로세스는 현재 동작하지 않으나 잔여 파일이 디스크 100% 유발
- 정리 후 56%로 복구 (197-199GB 확보)
- 다른 허브에도 동일 증상 있을 수 있음 → 전체 허브 디스크 사용량 점검 권장

### .env 유실 (15027, 15065, 15083, 15090)
- 원인 불명 (악성코드, 이전 작업 중 사고, 또는 초기 설치 미완료 가능성)
- **PM2 dump 파일(`~/.pm2/dump.pm2.bak`)에서 matterhub_id + hass_token 복구 성공** (4대 모두)
- 15065는 PM2 dump에 값 없었으나 PM2 로그 + bash history에서 복구

### .pyc 캐시 손상 (15065)
- `wifi_config/service.pyc` 파일 손상 → `EOFError: marshal data too short`
- API 서비스 60,000회 이상 크래시 반복 (restart counter=60406)
- `find . -name __pycache__ -exec rm -rf {} +`로 정리 후 정상 기동
